Das Ziel des Programs besteht darin, den Vorbereitungsklässer:innen und Neunteklässer:innen Verben mit Prepositionen
beizubringen. 

In dieser Datei befinden sich zwei unterschiedliche Versionen von dem selben Program.
Die erste Version funktioniert mit einer Csv (Comma seperated Values) Datei und Python. Obwohl sie komplett dieselbe Verben enthält,
soll diese Version mit Terminal aktiviert werden, daher fand ich diese Version des Programs ungenügend. (gelöscht wegen der Webseite)

Die zweite Version hingegen kann mit einem Klick eingesetzt werden. Sie funktioniert mit Html und ehrlich gesagt ist sie in jeder Hinsicht besser 
als die erste Version. (Die erste Version läuft theoretisch schneller aber wirklich egal.) Bei dieser Version legte ich die originale Python Version zugrunde
und ich visualisierte es.
